## Block file types

Block the upload or download of files based on their type.

| Selector           | Operator | Value                                   | Logic | Action |
| ------------------ | -------- | --------------------------------------- | ----- | ------ |
| Upload File Type   | in       | _Microsoft Office Word Document (docx)_ | And   | Block  |
| Download File Type | in       | _PDF (pdf)_                             |       |        |
