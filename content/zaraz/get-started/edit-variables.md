# Edit a variable

1. Log in to the [Cloudflare dashboard](https://dash.cloudflare.com/login), and select your account and domain.
2. Select **Zaraz** > **Variables**.
3. Locate the variable you want to edit, and select **Edit** to make your changes.
4. Select **Save** to save your edits. 

## Delete a variable

1. Log in to the [Cloudflare dashboard](https://dash.cloudflare.com/login), and select your account and domain.
2. Select **Zaraz** > **Variables**.
3. Locate the variable you want to delete, and select **Delete**.

{{<Aside type="warning" header="Important">}}
You cannot delete a variable that is being used by a tool or a trigger. If you want to delete a variable, you need to delete it first from the tool or trigger it is being used on.
{{</Aside>}}
