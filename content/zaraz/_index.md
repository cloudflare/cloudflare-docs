{{<beta>}} Cloudflare Zaraz {{</beta>}}

{{<description>}}
Offload third-party tools and services to the cloud and improve the speed and security of your website.
{{</description>}}

{{<plan type="all">}}

Cloudflare Zaraz (beta) gives you complete control over third-party tools and services for your website, and allows you to offload them to Cloudflare's edge, improving the speed and security of your website. With Cloudflare Zaraz you can load tools such as analytics tools, advertising pixels and scripts, chatbots, marketing automation tools, and more, in the most optimized way.

Cloudflare Zaraz is built for speed, privacy, and security, and you can use it to load as many tools as you need, with a near-zero performance hit.

---

## Features

{{<feature header="Tools support" href="/zaraz/get-started/add-tool/">}}
You can add many third-party tools to Zaraz, and offload them from your website.
{{</feature>}}

{{<feature header="Web API" href="/zaraz/web-api/">}}
Zaraz provides a client-side web API that you can use anywhere inside the `<body>` tag of a page.
{{</feature>}}

{{<feature header="Consent management" href="/zaraz/consent-management/">}}
Zaraz provides a Consent Management platform to help you address and manage required consents.
{{</feature>}}
