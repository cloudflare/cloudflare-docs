# Validation options

All certificates issued by Cloudflare - [Universal](/ssl/edge-certificates/universal-ssl/), [Advanced](/ssl/edge-certificates/advanced-certificate-manager/), and [Custom Hostname](/cloudflare-for-platforms/cloudflare-for-saas/security/certificate-management/) - are Domain Validated (DV) certificates.

If you need Organization Validated (OV) or Extended Validation (EV) certificates, [upload a custom certificate](/ssl/edge-certificates/custom-certificates/).
