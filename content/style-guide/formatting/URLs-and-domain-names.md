# URLs and domain names

When referring to a URL in text, do not use the `http://` or `www` with the Cloudflare web address. Use the `www` only when referring to URLs outside of Cloudflare. When referring to [cloudflare.com](https://www.cloudflare.com/), use `cloudflare.com`.
