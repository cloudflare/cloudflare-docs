If you believe your API key might be compromised, you should change your API key:

1.  Log in to the [Cloudflare dashboard](https://dash.cloudflare.com).
2.  Go to **Profile**.
3.  Go to **API Tokens**.
4.  In the **API Keys** section, find your key.
5.  Click **Change**.
