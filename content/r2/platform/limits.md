# Limits

## Account plan limits

{{<table-wrap>}}

| Feature                 | Limit                     |
| ------------------------| ------------------------- |
| Bucket                  | 1000 buckets per account  |
| Data storage per bucket | Unlimited                 |
| Object size             | 50 GB per object          |

To increase these limits, contact your Cloudflare account team.
