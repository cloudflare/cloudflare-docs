# Limits

All Security Intelligence API endpoints will be counted against the monthly quota, including Page Shield endpoints.

| Free | 100 |
| Pro | 100 |
| Business | 100  |
| Enterprise | 2,500  |

| Cloudflare One Premier | 50,000  |
