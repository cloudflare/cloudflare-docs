# Limitations
| Name | Non-Enterprise | Enterprise | 
| --- | --- | --- | --- |
| Load balancers | 20 | 100 |
| Monitor intervals | 60s (min), 3600s (max) | 10s (min), 3600s (max) |
| Monitors | 1.5x the number of pools | 1.5x the number of pools | 
| Origin servers | 20 | 250 |
| Pools | 20 | 100 |
