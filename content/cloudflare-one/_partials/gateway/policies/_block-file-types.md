## Block file types

Block the upload or download of files based on their type.

| Selector           | Operator | Value                                 | Logic | Action |
| ------------------ | -------- | ------------------------------------- | ----- | ------ |
| Upload File Type   | in       | Microsoft Office Word Document (docx) | And   | Block  |
| Download File Type | in       | PDF (pdf)                             |       |        |
