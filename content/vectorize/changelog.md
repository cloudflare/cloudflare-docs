# Changelog

## 2023-09-25

### Vectorize now in open beta

Vectorize, Cloudflare's vector database, is [now in open beta](BLOG-LINK-HERE). Vectorize allows you to store and efficiently query vector embeddings from AI/ML models from [Workers AI](/ai/), [OpenAI](), and other embeddings providers or machine-learning workflows.

To get started with Vectorize, [see the guide](/vectorize/get-started.md).
