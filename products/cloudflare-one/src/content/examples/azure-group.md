# Azure® Group

Allow members of an Azure Group. The ID is the group UUID (_`id`_) in Azure:

```json
{
  "azureAD": {
    "id": "86773093-5feb-48dd-814b-7ccd3676ff50",
    "identity_provider_id": "ca298b82-93b5-41bf-bc2d-10493f09b761"
  }
}
```
