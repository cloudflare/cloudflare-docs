# Updating fields

## Update schema.json in Google Cloud Function
Manually add missing fields to _schema.json_ via the Cloud Function UI:  

Click **Edit**

Once fields have been added please click the **Deploy** button.

The list of all fields available in Enterprise Log Share can be found here:  https://developers.cloudflare.com/logs/log-fields#http-requests
