# Create scheduled events

When you want to customize the behavior of a waiting room — for example, updating the queuing method, total active users, or any [other property](/reference/configuration-settings) — for a specific period of time, set up a scheduled event.

Any properties set on the event will override the default property on the waiting room for the duration of the event.

<Aside type="note">

Only certain customers can support scheduled events with their waiting rooms. For more details, see our <a href="/plans">Plans</a> page.

</Aside>
