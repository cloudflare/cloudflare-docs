# Bots

A **bot** is a software application programmed to do certain tasks. Bots can be used for good (chatbots, search engine crawlers) or for evil (inventory hoarding, credential stuffing).

<Aside type="note" header="More information">

For more background, refer to [What is a bot?](https://www.cloudflare.com/learning/bots/what-is-a-bot/).

</Aside>

## Verified bots

Cloudflare also maintains an internal list of **Verified Bots** that are associated with search engine optimization (SEO), website monitoring, and more. The method for allowing or blocking verified bots depends on [your plan](/get-started).

For a partial list of verified bots and more details on Firewall Rules, refer to [How does Firewall Rules handle traffic from known bots?](https://developers.cloudflare.com/firewall/known-issues-and-faq#how-does-firewall-rules-handle-traffic-from-known-bots).
