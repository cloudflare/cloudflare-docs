Microsoft Report Message (not compatible)

Due to changes in the flow of submission messages, Microsoft no longer honors mail flow rules for the Microsoft Report Message button. Therefore, Cloud Email Security is not compatible with this tool anymore.

To learn more about what is happening, log in to your Microsoft account as an administrator, and review the communication from Microsoft Open external link .

If you need to submit phish samples or missed phish to Cloud Email Security, we recommend using the PhishNet for Office 365 integrated app instead.
