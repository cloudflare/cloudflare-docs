Browser Rendering

Browser automation for Cloudflare Workers.

The Workers Browser Rendering API allows developers to programmatically control and interact with a headless browser instance and create automation flows for their applications and products. Once you configure the service, Workers Browser Rendering gives you access to a WebSocket endpoint that speaks the DevTools Protocol Open external link . DevTools is what allows Cloudflare to instrument a Chromium instance running in the Cloudflare global network.

Use Browser Rendering to:

  • Take screenshots of pages.
  • Convert a page to a PDF.
  • Test web applications.
  • Gather page load performance metrics.
  • Crawl web pages for information retrieval.
