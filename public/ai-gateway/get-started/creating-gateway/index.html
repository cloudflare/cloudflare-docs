Creating your first AI Gateway

In this guide, you will learn how to create your first AI Gateway. You can create multiple gateways to control different applications.

​​ Prerequisites

Before you get started, make sure you have a Cloudflare account Open external link .

​​ Setup

  • Dashboard
  • API

To set up an AI Gateway in the dashboard:

  1. Log into the Cloudflare dashboard Open external link and select your account.
  2. Go to AI > AI Gateway.
  3. Select Create Gateway.
  4. Enter your Gateway name and URL slug.
  5. Select Create.

To set up an AI Gateway using the API:

  1. Create an API token with the following permissions:
  • AI Gateway - Read
  • AI Gateway - Edit
  1. Get your Account ID.
  2. Using that API token and Account ID, send a POST request Open API docs link to the Cloudflare API.
